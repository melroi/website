Configuration du Point d’Accès WiFi D-Link DAP-1360


1. Connexion initiale

Réinitialiser le point d’accès WiFi en maintenant le bouton reset 10 à 15 secondes.

Brancher le DAP-1360 à une prise électrique et relier un câble Ethernet entre un PC et le port LAN du point d’accès.

Ouvrir un navigateur et accéder à l’interface via l’adresse :

http://192.168.0.50

Choisir Access Point comme mode d’opération si ce n’est pas déjà sélectionné.

Cliquer sur Manual Wireless Network Setup pour une configuration avancée.


2. Paramétrage IP

Pour le paramétrage IP, il faut suivre les captures d'écran :


3. Paramétrage sécurité

Pour le paramétrage sécurité, il faut aller dans l'onglet :

ADVANCED → ADVANCED WIRELESS


Finalisation

Sauvegarder la configuration.

Redémarrer l’appareil.

Reconnecter le point d’accès au réseau via un port du switch appartenant au bon VLAN.
