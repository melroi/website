Configuration de ProFTPD (FTP)


1ère étape : Installation de ProFTPD


On saisit les commandes suivantes :


apt update

apt install proftpd -y


Si l’installateur demande de choisir un mode, sélectionner le mode autonome (standalone mode).


2ème étape : Configuration de base


Dans le fichier /etc/proftpd/proftpd.conf, nous avons modifié les lignes suivantes :


  • AllowForeignAddress on → Autorise les connexions depuis des adresses IP non locales.
  • RootLogin on → Permet les connexions au compte root via FTP.
  • DefaultRoot ~ → Restreint chaque utilisateur à son répertoire personnel.

Ensuite, redémarrer le service avec la commande :

systemctl restart proftpd


3ème étape : Création d’un utilisateur FTP


On tape la commande suivante :


adduser rhftp


Un dossier personnel est automatiquement créé : /home/rhftp
