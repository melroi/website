Installation Windows 11 (Client)

  1. Démarrage sur la clé USB :
    • Allumer le PC.
    • Appuyer plusieurs fois sur F12 (ou F10/ESC selon la carte mère).
    • Sélectionner la clé USB avec mention UEFI puis appuyer sur Entrée.
  2. Écran d’installation Windows :
    • Choisir la langue puis cliquer sur Suivant.
    • Cliquer sur Installer maintenant.
    • Si demandé, cliquer sur "Je n'ai pas de clé de produit".
    • Choisir Windows 11 Pro.
    • Accepter les termes et cliquer sur Suivant.
  3. Erreur GPT ("Windows ne peut pas être installé…") :
    • Appuyer sur Shift + F10 pour ouvrir l’invite de commande.
    • Taper : diskpart → list disk → select disk 0 → clean → convert gpt → exit.
  4. Création de la partition :
    • Cliquer sur le disque vide, Nouveau → Appliquer → OK.
    • Sélectionner la partition principale la plus grande et cliquer sur Suivant.
  5. Installation automatique : Laisser Windows copier les fichiers (5-15 min).
  6. Réglage du boot au redémarrage :
    • Appuyer sur F12 ou F10, Boot Settings, Windows Boot Manager en priorité, sauvegarder (F10).
  7. Bypass NRO / Réseau :
    • Appuyer Shift + F10, taper oobe\bypassnro, redémarrage automatique.
    • Cliquer sur “Je n’ai pas Internet” → Continuer avec configuration limitée.
  8. Création de l’utilisateur : client1, mot de passe simple, questions sécurité aléatoires.

Installation Windows Server 2022

  1. Boot sur la clé USB :
    • Redémarrer le PC.
    • Appuyer sur F12 ou F10.
    • Sélectionner UEFI : clé USB puis appuyer sur Entrée.
  2. Démarrage de l’installation :
    • Langue / clavier → Suivant.
    • Cliquer sur Installer maintenant.
    • Choisir Windows Server 2022 Standard (expérience utilisateur).
    • Pas de clé → Suivant → accepter conditions.
    • Sélectionner Installation personnalisée.
  3. Partition (GPT recommandé) :
    • Cliquer sur la partition.
    • Nettoyer avec Shift + F10 → diskpart → clean → convert gpt.
    • Cliquer sur Suivant.
  4. Installation automatique, redémarrage automatique après installation.
  5. Mot de passe administrateur : définir et confirmer un mot de passe sécurisé.
  6. Configuration IP :
    • Panneau de configuration → Réseau → Modifier adapteur.
    • Clic droit Ethernet → Propriétés → IPv4 → Propriétés.
  7. Installer rôles : Active Directory, DHCP, DNS, Web Server (IIS).
  8. Boot Manager : priorité Windows Boot Manager (F10/F12).

Installation de Debian 12 - Guide détaillé

  1. Démarrage sur la clé USB :
    Allumez le PC puis appuyez immédiatement et plusieurs fois sur F12 ou F10 pour accéder au menu de démarrage.
    Dans le menu qui s'affiche, sélectionnez l'entrée UEFI : [Nom de votre clé USB] puis appuyez sur Entrée.
    Choisissez ensuite l'option Install Graphique dans le menu Debian.
  2. Configuration de base :
    Sélectionnez :
    • Langue : Français
    • Pays : France
    • Clavier : Français
  3. Nom de la machine et domaine :
    • Nom d’hôte (hostname) : srv-debian
    • Domaine : patate.local
  4. Utilisateurs et mots de passe :
    • Mot de passe root (administrateur complet)
    • Créer un utilisateur classique, par exemple
    • Définir un mot de passe fort pour cet utilisateur
  5. Partitionnement :
    Choisissez Assisté - utiliser un disque entier avec schéma LVM (Volume Logique) pour une gestion simplifiée des partitions.
    Confirmez le disque à utiliser (ex: /dev/sda) et validez les changements proposés.
  6. Sélection des paquets logiciels :
    À l’étape du choix de l’environnement de bureau :
    • Sélectionnez : GNOME
  7. Installation du chargeur de démarrage (GRUB) :
    Lorsqu’on vous demande où installer GRUB, choisissez /dev/sda (ou le disque principal).
    Cela rend Debian amorçable au démarrage.
  8. Configuration du Boot Prioritaire :
    Une fois l’installation terminée, redémarrez le PC.
    Appuyez sur F12 ou F10 pour accéder au menu de démarrage.
    Allez dans les Boot Settings (ou ordre de démarrage) et placez Debian / GRUB comme premier choix.
    Sauvegardez avec F10.

Présentation de Webmin

  • Webmin est une interface web permettant d’administrer un serveur Linux à distance, sans avoir besoin de passer par la ligne de commande. Il offre une gestion graphique des services (FTP, DHCP, etc).
