Configuration des utilisateurs et des groupes


1ère étape : Création du domaine

Gérer → Ajouter des rôles → Installation basée sur un rôle ou une fonctionnalité → Cocher AD DS et Serveur DNS → Ajouter les fonctionnalités → Installer

Une fois l’installation terminée : Drapeau → Promouvoir ce serveur en contrôleur de domaine → Ajouter une nouvelle forêt → Nom du domaine : patate.local → Mot de passe : Patate1234 → Installer


2ème étape : Création des utilisateurs et des groupes

Outils → Utilisateurs et ordinateurs Active Directory → Clic droit sur le domaine → Nouveau → Utilisateur


3ème étape : Ajout des deux clients au domaine

Panneau de configuration → Système et sécurité → Système → Paramètres de nom de l’ordinateur, domaine et groupe de travail → Modifier → Entrer Patate.local → Se connecter avec un utilisateur du domaine


4ème étape : Mise en œuvre de stratégies de sécurité

Outils → Gestion de stratégies de groupe → Sélectionner le domaine ou un groupe → Clic droit → Créer un GPO


5ème étape : Création d’un dossier personnel et d’un lecteur réseau

Explorateur de fichiers → Ce PC → Disque local → Créer un dossier partagé → Propriétés → Partage → Partager


Créer ensuite un dossier spécifique pour chaque utilisateur dans ce dossier partagé, lui attribuer les droits de modification et d’écriture.
