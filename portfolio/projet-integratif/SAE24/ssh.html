Configuration du service SSH sur le serveur Linux


1ère étape : installation du service SSH

On commence par désinstaller le service SSH avec la commande :

apt remove --purge openssh-*

On efface le fichier ~/.ssh/known_hosts avec la commande :

rm ~/.ssh/known_hosts

Puis on le réinstalle avec :

apt install openssh-*


2ème étape : test

Après avoir installé le service SSH, on teste si l’on peut se connecter au serveur via un autre appareil. Pour cela, on tape sur un autre PC du réseau avec la commande :

ssh -p 2222 rh@172.25.0.135

rh = le nom d’utilisateur

172.25.0.135 = adresse sur laquelle on se connecte (port du routeur qui fait le NAT)

-p 2222 = le port sur lequel on se connecte sur le routeur (le port 22 ne fonctionne pas)
