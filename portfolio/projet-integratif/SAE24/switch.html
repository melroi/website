Configuration des vlans

1ère étape : Création des VLANs

On tape les commandes suivantes pour créer et donner un nom aux différents VLANs du réseau :

enable => conf t => vlan 10 => name Finance

enable => conf t => vlan 20 => name RH

enable => conf t => vlan 30 => name Public

enable => conf t => vlan 40 => name Admin


Pour le VLAN Finance (ports 1 à 10) :

Exemple pour le port 1 :

interface GigabitEthernet1/0/1

switchport mode access

switchport access vlan 10
